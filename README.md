# ML_Sys_Design_Project


## Умный генератор коммерческих предложений на основе LLM
### Суть проекта:
Разработать AI-сервис, который автоматически формирует коммерческие предложения (КП) на основе входящих заявок от клиентов — будь то письмо, PDF, скан или текстовый файл. Сервис будет использовать LLM/VLLM для анализа запроса, извлечения ключевых параметров и генерации корректных наименований продукции в соответствии с внутренней документацией, спецификациями поставщиков и системой бизнес-правил.
